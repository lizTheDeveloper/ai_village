How It Works

This demo uses a SharedWorker to run the game simulation independently from the browser windows.

Try this: Open this page in multiple tabs. Notice how they all show the same synchronized simulation state!

The simulation runs at 20 TPS in the worker, and all windows receive state updates every tick.

Even if you close all windows and reopen them, the simulation persists in IndexedDB and continues where it left off.
